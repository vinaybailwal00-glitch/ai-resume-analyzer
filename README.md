# AI Resume Analyzer

An AI-powered resume screening application built with Python and Streamlit.

## Features

- Upload Resume PDF
- ATS Match Score
- Job Description Analysis
- Missing Keyword Detection
- Resume Improvement Suggestions

## Tech Stack

- Python
- Streamlit
- PyPDF2
